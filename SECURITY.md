# Security

This repository hosts **documentation only** (Markdown and community templates). It does not ship runnable application code for the Aippy platform.

## Reporting issues

- **Documentation errors or sensitive data accidentally committed:** open a private security advisory via GitHub **Security → Advisories** for this repository, or contact maintainers through the channel they list in the repository description.
- **Vulnerabilities in the Aippy product** (aippy.ai): report through the **official** security contact or support channels published by Aippy — not through this docs repo unless maintainers explicitly accept such reports.

Please do not open public issues for undisclosed sensitive information.
